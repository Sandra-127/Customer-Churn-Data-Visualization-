# Customer Churn Analysis & Data Visualization

##  Internship Project

Completed as part of a **Data Visualization Internship at SaiKet Systems (July–August 2025)**.

---

## Project Objective

Analyze customer churn data from a telecom company and create visual insights to:

* Identify churn patterns
* Understand customer behavior
* Support business decision-making

---

##  Dataset

* Telco Customer Churn Dataset
* Includes:

  * Tenure
  * Monthly Charges
  * Gender
  * Churn status

---

##  Tasks Performed

### Task 1: Data Overview & Sorting

* Explored dataset structure
* Sorted by tenure
* Filtered churned customers

### Task 2: Churn Count Visualization

* Created column chart for churn vs non-churn

### Task 3: Monthly Charges Distribution

* Built histogram
* Analyzed charge patterns

### Task 4: Churn Rate by Gender

* Used Pivot Table
* Created clustered bar chart

### Task 5: Heatmap Analysis

* Visualized relationship between:

  * Monthly Charges
  * Tenure
  * Churn

---

## Key Insights

* Customers with shorter tenure show higher churn
* Monthly charges influence churn probability
* Gender-based differences observed in churn trends

---

##  Tools Used

* Microsoft Excel
* Data Visualization Techniques
* Pivot Tables & Charts
